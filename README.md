# 🎮 Rock Paper Scissors Game

A fun and interactive **Rock ✊ Paper ✋ Scissors ✌️** game built using **Python (PySide6)** with a simple and modern graphical interface.  
The game lets users play against the computer, tracks scores, and displays dynamic results in real time.

---

## 🚀 Features

- 🧠 **Smart Backend Logic** – Randomized computer moves with fair win conditions.  
- 🧑‍💻 **Interactive UI** – Built with PySide6 for smooth user experience.  
- 🧾 **Score Tracking** – Keeps track of both player and computer scores.  
- 🔄 **Reset Anytime** – Reset the tour without restarting the app.  
- 🏆 **Game Over Summary** – Displays final winner and scores when you exit.

---

## 🛠️ Technologies Used

- **Python 3.10+**
- **PySide6 (Qt for Python)**
